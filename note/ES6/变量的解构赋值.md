默认值
解构赋值允许指定默认值
注：ES6 使用严格相等运算符，判断一个位置是否有值。所以只有当一个数组成员严格等于undefined，默认值才会生效。

```
let [x = 1] = [undefined];
x // 1

let [x = 1] = [null];
x // null
```
如果一个数组成员事null，默认值就不会生效，因为null不严格等于undefined。

+ 字符串的解构赋值

		```
		const [a, b, c, d, e] = 'hello';
		```
+ 函数参数得解构赋值
```
	[[1, 2], [3, 4]].map(([a, b]) => a + b);
```


用途
1. 交换变量得值
2. 从函数返回多个值
3. 函数参数的定义
4. 提取JSON数据
5. 函数参数的默认值
6. 遍历Map结构
7. 输入模块的指定方法
